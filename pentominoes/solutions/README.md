# Pentomino puzzle solutions

Folders of pentomino puzzle solutions by board size. The file names are the solution numbers used by [Isomer Design](https://isomerdesign.com/Pentomino/).
The solutions were generated by a randomized method, the latest version of which is [experiments.Interchange](https://github.com/knoxa/automata/blob/main/src/experiments/Interchange.java).
